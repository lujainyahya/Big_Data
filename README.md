# Employee Compensation Analysis (Big Data Project)

This project focuses on analyzing employee compensation data using data analysis and machine learning techniques. The goal is to understand the factors that influence employee salaries and benefits, and to build predictive models for total compensation.

## Project Overview
Employee compensation is an important indicator for organizations to understand workforce costs and improve human resource decision-making. In this project, we analyze a dataset containing employee salary components such as base salary, overtime pay, retirement contributions, and health benefits.

Using data preprocessing and machine learning techniques, we aim to uncover patterns in the data and predict total employee compensation.

## Objectives
- Analyze employee compensation data.
- Identify patterns and relationships between salary components.
- Build machine learning models to predict total compensation.
- Improve understanding of factors affecting employee pay.

## Dataset
The dataset used in this project contains information about employee compensation, including:
- Base salary
- Overtime pay
- Other salaries
- Retirement benefits
- Health benefits
- Total compensation

(Source: Kaggle)

## Data Preprocessing
Several preprocessing steps were applied to prepare the data for analysis:
- Handling missing values using mean imputation and KNN Imputer.
- Removing outliers using the IQR method.
- Feature transformation and normalization.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Data preprocessing techniques
- Machine learning algorithms

## Project Goal
The main goal of this project is to demonstrate how data analysis and machine learning can be used to extract insights from employee compensation data and support better organizational decision-making.
